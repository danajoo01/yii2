YII2 Stok Barang
============================

### Instalasi

Sebelum melakukan instalasi pastikan sudah terpasang `composer` dan `php`. 

Requitment php min versi 7.4

```php
// Setup library yang dibutuhkan
$ composer install

$ yii migrate

```
